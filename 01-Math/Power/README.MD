# Power of a Number

## Problem
Given two integers `base` and `exponent`, compute:

base ^ exponent

That is, multiply the base number by itself `exponent` times.

---

## Examples
Input:  base = 2, exponent = 3
Output: 8

Input:  base = 5, exponent = 0
Output: 1

Input:  base = 3, exponent = 4
Output: 81
